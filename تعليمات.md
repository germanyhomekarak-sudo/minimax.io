# 🛠 al BOURJI ELECTRICS — التحديث

## ✨ إيش الجديد؟

### 1. حفظ تلقائي فوري 🔄
- كل ما تغيّرين سعر/منتج/نص → **ينحفظ تلقائياً** بدون ما تضغطين شي
- تشوفين إشعار أخضر "✓ تم الحفظ تلقائياً" بأسفل الشاشة
- لما ترجعين للموقع لاحقاً → **تلقائياً** تشوفين تعديلاتك

### 2. لوحة المطور مخفية تماماً 🔒
- ❌ **ما عاد في زر مطور ظاهر** للزوار
- ✅ **4 طرق سرية** لفتحها (اختاري وحدة):

#### الطريقة 1: اختصار لوحة المفاتيح (الأسهل)
اضغطي: **Ctrl + Shift + D** (ويندوز/لينكس) أو **Cmd + Shift + D** (ماك)

#### الطريقة 2: Konami Code (الكلاسيكي 🎮)
اضغطي بالترتيب:
⬆️ ⬆️ ⬇️ ⬇️ ⬅️ ➡️ ⬅️ ➡️ **B A**

#### الطريقة 3: نقر 7 مرات سريعة على الشعار
اضغطي 7 مرات على اسم البراند أو الشعار في أعلى الصفحة (خلال ثانيتين)

#### الطريقة 4: من الـ Console (للمطور)
1. افتحي Developer Tools (F12)
2. روحي لـ Console tab
3. اكتبي:
```js
__alBurjiOpenDev()
```

### 3. كود الدخول ما تغيّر
الكود باق نفسه: **`9p3cas99`**

## 📁 الملفات المعدّلة
- `index.html` — إخفاء زر المطور
- `js/main.js` — إضافة دالة التحديث الفوري
- `js/dev.js` — حفظ تلقائي + 4 طرق فتح سرية

## 🚀 كيف ترفعين التحديث؟
1. حمّلي الملفات الثلاثة من الأسفل
2. ارفعيهم على GitHub في نفس المسار:
   - `index.html` → الجذر
   - `js/main.js` → مجلد `js/`
   - `js/dev.js` → مجلد `js/`
3. انتظري دقيقة → التحديث يظهر تلقائياً على `germanyhomekarak-sudo.github.io/minimax.io/`
